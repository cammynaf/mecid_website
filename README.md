# mecid_website
Website for Mengo Educational Community and Inclusive Development (MECID) organization
## Quick publish on GitHub Pages
1. Create a new GitHub repo (public), e.g. `mecid-website`.
2. Upload `index.html` (and this README, optional) to the repo.
3. Go to **Settings â†’ Pages**.
4. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Save. GitHub will provide your site link.

## Edit placeholders
Open `index.html` and replace:
- email, phone
- area of operation (counties/sub-counties)
- website link text in the Contact section
